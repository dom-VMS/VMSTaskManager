# VMS Task Manager

A web application designed for Varland Metal Service employees to file, view, and updates tasks.

Technical Details

* Ruby version - 2.3.7
* Rails version - 5.2.1
* MySQL version - 8.0.12 Homebrew

* Gems
  - Carrierwave
  - Bootstrap
  - JQuery
  - Fontawesome-rails
  - Bcrypt
